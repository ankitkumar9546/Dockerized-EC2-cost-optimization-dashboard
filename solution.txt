To solve this problem, I developed a Cloud Cost Optimizer Dashboard using Streamlit, Boto3, Docker, and AWS EC2. The dashboard provides a simple and interactive interface to monitor EC2 instances, estimate cloud costs, and view infrastructure status in real time. The application was containerized using Docker and deployed on AWS EC2 with IAM Role-based authentication for secure cloud access. The project focuses on making AWS monitoring more accessible, lightweight, and practical for learners and small cloud environments.
